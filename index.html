YarnLang

A programming language for crochet word tapestries

GitHub

How It Works

Use keywords "make:", "word:", "object:", and "color:" to specify what you want to make. You can either specify colors or use the default black and white given by YarnLang. After running the program the interpreter will output a simulation of the tapestry and give the simple pattern directions for the tapestry(instructions not available for object tapestries).

Supported colors: black, white, red, green, yellow, blue, magenta, cyan, light gray, dark gray, bright red, bright green, bright yellow, bright blue, bright magenta, and bright cyan. Words tapestries only support letters and available object tapestries are flower, bow, and star.

Example Tapestry and Steps

1. From github download the given yl_interpreter.py file and sample program 1 which contains a blank template
2. Edit sample program to what you want, you can do multiple tapestries within the same program!
3. In your terminal type "python3 yl_interpreter.py program1.yl"
4. See the tapestry come to live!
