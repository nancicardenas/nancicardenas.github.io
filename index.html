YarnLang

A programming language for crochet word tapestries

How It Works

Use keywords "make:", "word:", "object:", and "color:" to specify what you want to make. You can either specify colors or use the default black and white given by YarnLang. After running the program the interpreter will output a simulation of the tapestry and give the simple pattern directions for the tapestry.
